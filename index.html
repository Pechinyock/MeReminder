Visual studio

Shortcuts

  • ctrl + q -- settings search
  • f12 -- go to symbolc
  • ctrl + - -- step back
  • ctrl + shift + backspace -- back to start
  • ctrl + g -- go to line
  • ctrl + [, ctrl + s -- focus file in solution explorer
  • ctrl + tab -- choose one of opened tab
  • ctrl + shift + v -- get value of copy buffer
  • ctrl + u -- to lower case
  • ctrl + shift + u -- to upper case
  • ctrl + alt + mouse left click -- add text cursor
  • ctrl + k, ctrl + c -- comment block code
  • ctrl + k, ctrl + u -- uncomment block
  • ctrl + k, ctrl + s -- warp code block with ...
  • ctrl + . -- open 'quic actions and refactoring'
  • ctrl + r, ctrl + r -- smart rename
  • ctrl + b -- build project
  • ctrl + alt + o -- open output window
  • alt + mouse selet -- get rectangle to seletc
  • ctrl + k, ctrl + o -- toggle between .h file and .cpp file
